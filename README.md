# Multiplayer Tic-Tac-Toe

A real-time multiplayer web application that enables two players to compete seamlessly through their browsers. The project demonstrates real-time communication using WebSockets and a responsive user interface for an engaging gameplay experience.

## Features

* Real-time multiplayer gameplay
* Room-based matchmaking
* Live game state synchronization
* Win, draw, and score tracking
* Responsive design for desktop and mobile

## Tech Stack

* Python
* Flask
* Flask-SocketIO
* HTML
* CSS
* JavaScript
